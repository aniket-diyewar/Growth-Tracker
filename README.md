# 📚 Growth Tracker — StudyStack

A mobile productivity app for engineering students to track daily study routines, DSA practice, aptitude prep, and semester studies.

## Features
- ✅ Daily Task Tracker (Aptitude / DSA / Semester)
- ⏱ Focus Timer (10, 20, 30, 40 min presets)
- 📊 Progress Tracker with weekly calendar
- 📅 6-day Weekly Study Plan
- 🔥 Streak Tracker & Motivation System
